How High-Amplitude Outliers Create a Widening Rolling RMS Pattern

This is a small synthetic example. The baseline vibration stays constant. Only the number of sparse high-amplitude outliers changes. As the outlier population increases, rolling RMS develops large peaks at increasingly separated outlier clusters.

RMS_window = sqrt((sum(clean_samples^2) + sum(outlier_samples^2)) / N)
Approximation used here: RMS_window = sqrt((N * baseline_rms^2 + k_outliers * A_outlier^2) / N)
Winsorized version: RMS_window = sqrt((N * baseline_rms^2 + k_outliers * min(A_outlier, cap)^2) / N)
